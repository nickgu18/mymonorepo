# Plan-NG Skill

This skill provides a structured template for analyzing tasks and formulating a plan.

## How to use

1. Read the `template.md` file located in this skill's directory (`/usr/local/google/home/guyu/Desktop/mymonorepo/.gemini/skills/plan-ng/template.md`).
2. Fill out the sections of the template based on the issue or task at hand:
   - **task**: A brief summary of what needs to be done.
   - **constraints**: Any technical or business limitations.
   - **definition of done**: Clear criteria for when the task is considered complete.
   - **fix_plan**: Step-by-step plan to achieve the definition of done.
   - **files**: List of files that need to be modified or created.
3. Output the filled template as the analysis report.
