### analysis

Iteration 4 never produced founder-level experience aggregates (role-type counts and duration) from `founder_experience[_training].csv`; experience is only used to pick the last founded company and to build performance/network/team features, so the model sees no explicit experience features and there is no time-respecting experience snapshot keyed by a per-founder `cut_off_year`.

### fix_plan

- Add a new helper `compute_founder_experience_features` in `xgboost/src/features/feature_factory.py` that takes the cleaned founder experience table plus an optional mapping `cut_off_years: Mapping[str, int]` (person_id → inclusive snapshot year) and a fallback `current_year`, and returns one row per founder with columns: `person_id`, `n_c_suite_roles`, `n_employee_roles`, `n_executive_roles`, `n_founder_roles`, `n_board_roles`, `n_experience_roles`, `total_experience_years`.
- Algorithm 1 (IEEE style) — `compute_founder_experience_features`:

```text
Algorithm 1: compute_founder_experience_features
Input:
    E  : founder experience table with columns
         [person_id, start_date, end_date,
          is_c_suite, is_employee, is_executive,
          is_founder, is_on_board]
    C  : optional mapping person_id → cut_off_year (integer)
    Yd : default snapshot year (integer, e.g. current_year)
Output:
    F  : table with one row per person_id and columns
         [person_id,
          n_c_suite_roles, n_employee_roles,
          n_executive_roles, n_founder_roles,
          n_board_roles, n_experience_roles,
          total_experience_years]

1:  Start
2:  Parse E.start_date, E.end_date to datetime;
    coerce all role flags to {0, 1} with missing → 0
3:  F ← empty set
4:  For each unique person_id p in E do
5:      Rp ← { rows r in E where r.person_id = p }
6:      if C is defined and p ∈ C then
7:          cut_year ← C[p]
8:          Rp ← { r in Rp | year(r.start_date) ≤ cut_year }
9:      end if
10:     if Rp is empty then
11:         Append to F a row
           [p, 0, 0, 0, 0, 0, 0, 0.0] and continue
12:     end if
13:     Initialize counters for p:
           n_c_suite ← 0; n_employee ← 0; n_exec ← 0;
           n_founder ← 0; n_board ← 0;
           n_roles ← 0; total_years ← 0.0
14:     For each row r in Rp do
15:         if r.start_date is null then
16:             continue
17:         end if
18:         if r.end_date is null then
19:             if C is defined and p ∈ C then
20:                 effective_end ← 31-Dec-cut_year
21:             else
22:                 effective_end ← 31-Dec-Yd
23:             end if
24:         else
25:             effective_end ← r.end_date
26:             if C is defined and p ∈ C and
                year(effective_end) > cut_year then
27:                 effective_end ← 31-Dec-cut_year
28:             end if
29:         end if
30:         duration_years ← max((effective_end − r.start_date).days / 365.25, 0)
31:         total_years ← total_years + duration_years
32:         n_roles ← n_roles + 1
33:         n_c_suite ← n_c_suite + r.is_c_suite
34:         n_employee ← n_employee + r.is_employee
35:         n_exec ← n_exec + r.is_executive
36:         n_founder ← n_founder + r.is_founder
37:         n_board ← n_board + r.is_on_board
38:     end for
39:     Append to F a row
        [p, n_c_suite, n_employee, n_exec,
            n_founder, n_board, n_roles, total_years]
40: end for
41: Return F
```
