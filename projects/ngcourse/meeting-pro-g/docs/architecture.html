Component Details

  • AppScript (AS): "Sensor" & "Authorizer". Runs as User. Fetches emails & tokens.
  • Cloud Run (ADK): "Brain". Stateless. Executes Agentic Logic.
  • Agent Logic:
    • Root: Router.
    • Email Triage Agent: Classifies intent (Schedule, Confirm, Cancel, Unknown).
    • Scheduling Agent: Execute calendar logic (Read/Write) based on intent.
    • Author Agent: Polishes tone and creates the final Gmail draft.
  • BigQuery: "Memory". Logs request lifecycle.
