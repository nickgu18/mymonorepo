# Debrief Skill

This skill guides the process of running a debrief at the end of a session or task to capture learnings.

## Workflow

1. **Locate Template:** Find the hotwash template at `.learnings/templates/hotwash.md`.
2. **Analyze Session:** Review the activities, successes, and challenges of the current session or task.
3. **Draft Report:** Populate the hotwash template with:
   - Top 3 Strengths/Successes
   - Top 3 Areas for Improvement
   - General Comments/Remarks
4. **Save Entry:** Save the completed report to `.learnings/entries/` with a descriptive filename (e.g., `YYYY-MM-DD_task_name_hotwash.md`).
