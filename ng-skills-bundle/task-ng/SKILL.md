# Task-NG Skill

This skill provides instructions for tracking task progress systematically.

## How to use

1. Create or locate a `uuid_task.md` file in the `/usr/local/google/home/guyu/Desktop/mymonorepo/.scratch_space` directory.
2. Maintain a markdown checklist (`- [ ]` and `- [x]`) of the subtasks.
3. Update the checklist by checking off items as you complete them.
4. Consult the list to determine the next step in your workflow.
