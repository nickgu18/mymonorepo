# Forest Skill

Use this skill when creating interfaces in the **Forest** style.

## Goal
Build UI screens that follow the style rules in `design-system.md` as the authoritative source.

## Instructions
1. Read `design-system.md` first and follow its color, typography, spacing, and interaction rules.
2. Keep component visuals and motion consistent with the design philosophy.
3. Prefer reusable tokens and component patterns over one-off styling.

## Deliverables
- Production-ready UI implementation
- Any required theme tokens / CSS variables
- Components aligned with the design system
